# WP5: Transparent AI, (Augmented Reality)* and Visual Analytics
This folders contains the software developed in WP5 ragarding the Explainable AI and Visual Analytics components of the project. In detail, it contains:
- The main code of the LORE (Local Rule-based Explanations) framework for explainable AI, along with its documentation and examples.
- The set of custom neighborhood generators developed for LORE to handle different data types and scenarios coming from WP2.
- The code for the visual analytics dashboard that provides an interactive interface for exploring and understanding rules generated by a model
