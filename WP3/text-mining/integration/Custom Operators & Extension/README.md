
# Custom Operators & Extension
T4.5 text mining RM integration in CREXDATA.

## Contents
Contains sample RapidMiner processes and RapidMiner AI Hub and custom operators.
  - *AI Hub Endpoints*: RM processes to create endpoints on AI Hub.
  - *Custom Operators*: contains custom operators.
  - *Custom Operators Workflows*: Workflows used to create custom operators.
  - *Custom Python Operators*: As python operators for the *Python Scripting* extension.
  - *Sample Data*: contains sample data for running RM processes.
  - *Sample Processes*: sample RM processes.
  - *Extension*: Extension contain all custom operators.

## requirements 
  - RapidMiner AI Studio Software 2024.1
  - RapidMiner AI Studio extensions:
    - *Generative AI*
    - *Generative Models*
    - *Python Scripting*
    - *Custom Operators*
    - *Text Processing*
    - *Operator Toolbox*
  - rm-genai python environment (follow [instructions](https://docs.rapidminer.com/2024.1/studio/generative-ai/index.html)), and then setup conda enviroment with rm_genai.yml.
